Project In Development (Horror Fishing)

In Development

See code breakdown

A horror survival game currently in a prototype phase. Set in the vast landscapes of Alaska, you must fish to survive the day and build to survive the night. Currently prototyping and defining project requirements before I determine the game engine. So far I have create the day/night cycle with scope to create it into a full 'Dynamic World System' that controls the time of day and weather throughout the game. My next step is to create a fishing system that allows the player to cast, reel, select bait, catch different fish species and the player gains different beneficial stats depending on the fish they eat or gains currency depending on the rarity of the fish. You can see how some of my day/night system works in the screenshots above.
